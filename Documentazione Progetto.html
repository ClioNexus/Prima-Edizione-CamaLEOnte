Manuale d'Uso - Il CamaLEOnte

Questa documentazione riassume le caratteristiche tecniche del progetto e le procedure per l'aggiornamento.

1. Funzioni Implementate

  • Sfoglio Turn.js: Simulazione realistica del passaggio di pagina.
  • Indice Interattivo: Hotspot cliccabili a pagina 3 per navigazione rapida.
  • Video Autoplay (Pag 23): Il video parte automaticamente al raggiungimento della pagina e si interrompe al cambio.
  • Google Analytics: Monitoraggio visualizzazioni attivo.

3. Come Aggiornare il Sito

Ogni volta che aggiungi un'immagine (come quella delle statistiche) o modifichi il codice, devi caricare i file su GitHub:

git add . git commit -m "Aggiunti screenshot statistiche" git push origin main
NOTA SULLE IMMAGINI: Assicurati che i file immagine siano nella stessa cartella di questo file HTML prima di fare il push.

4. Struttura dei File

  • index.html: Motore principale.
  • stats_analytics.png: Screenshot delle statistiche.
  • [1-24].png: Pagine del giornale.
  • favicon.png: Icona del sito.
