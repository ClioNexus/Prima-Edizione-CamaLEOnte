Manuale d'Uso - Il CamaLEOnte

Questa documentazione riassume le caratteristiche tecniche del progetto e le procedure per l'aggiornamento.

1. Funzioni Implementate

  • Sfoglio Turn.js: Simulazione realistica del passaggio di pagina.
  • Indice Interattivo: Hotspot cliccabili a pagina 3 per navigazione rapida.
  • Video Autoplay (Pag 23): Il video parte automaticamente al raggiungimento della pagina e si interrompe al cambio, senza barre di controllo browser.
  • Google Analytics: Monitoraggio visualizzazioni tramite ID inserito nelle righe 13 e 19.
  • Favicon: Icona personalizzata caricata tramite il file favicon.png.

2. Come Aggiornare il Sito

Dopo ogni modifica ai file locali, esegui questi comandi nel terminale di VS Code:

git add . git commit -m "Descrizione della modifica" git push origin main
ATTENZIONE: Dopo il push, il sito live potrebbe impiegare 2-3 minuti per aggiornarsi. Se non vedi le modifiche, premi CTRL + F5 nel browser.

3. Struttura dei File

  • index.html: Il motore principale del sito.
  • jquery.min.js & turn.min.js: Librerie per l'animazione.
  • [1-24].png: Le pagine della rivista.
  • 23.mp4: Il video della pagina 23.
  • favicon.png: L'icona del browser.
