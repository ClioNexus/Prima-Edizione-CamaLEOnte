Sicurezza e Setup Ambiente

Bonifica Git e isolamento file

Lancio su Cloud (GitHub)

Risoluzione conflitti e hosting

▼
Configurazione Remota: Collegamento all'URL HTTPS di GitHub tramite git remote add origin.
Gestione Conflitti: Risoluzione errore [rejected] main -> main (fetch first) causato dalla licenza MPL 2.0 creata su web ma non presente in locale.
Forzatura Push: Utilizzo di git push -f origin main per allineare forzatamente l'archivio locale a quello remoto.
Deployment Pages: Configurazione in Settings -> Pages per l'erogazione del sito statico dalla root del branch main.
